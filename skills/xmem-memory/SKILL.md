# XMem Memory

Use the XMem MCP tools whenever the user asks to remember, search, recall, or preserve context across sessions.

- Store durable memories only when they are useful beyond the current chat.
- Use project scope for repository workflows, architecture, design decisions, and build or test commands.
- Use user scope for stable personal preferences.
- Search or retrieve before answering questions that depend on previous sessions.
- Never save secrets, credentials, private keys, API keys, or one-time tokens.
